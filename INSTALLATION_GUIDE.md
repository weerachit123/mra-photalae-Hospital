# คู่มือการติดตั้ง MRA Audit System ในเครือข่ายโรงพยาบาล

ระบบ MRA ถูกพัฒนาให้ทำงานผ่าน Browser (เว็บ) แต่จำเป็นต้องมี **เครื่องคอมพิวเตอร์หลัก (Server) 1 เครื่อง** ภายในโรงพยาบาล เพื่อเป็นตัวกลางนำข้อมูลจาก HosXP ขึ้นมาแสดงผล

---

## ขั้นตอนที่ 1: สิ่งที่ต้องเตรียม
1. เครื่องคอมพิวเตอร์ระบบปฏิบัติการ Windows (PC ธรรมดา หรือ Server ก็ได้)
2. เครื่องนี้ต้อง **มองเห็น/Ping เจอ 192.168.0.5** (เชื่อมต่อเครือข่ายเดียวกันกับฐานข้อมูล HosXP)
3. เชื่อมต่ออินเทอร์เน็ต (ในครั้งแรกที่มีการใช้ไฟล์ติดตั้ง)

## ขั้นตอนที่ 2: ติดตั้งโปแกรมพื้นฐาน (Node.js)
ระบบ MRA ขับเคลื่อนด้วยโปรแกรมชื่อ `Node.js` หากเครื่องคุณไม่มี ให้ทำการติดตั้งก่อน
1. เข้าไปที่เว็บ: [https://nodejs.org/en/](https://nodejs.org/en/)
2. กดปุ่มดาวน์โหลดตัว **LTS** (แนะนำสำหรับผู้ใช้ทั่วไป) 
3. เมื่อโหลดเสร็จ ดับเบิลคลิกไฟล์ติดตั้ง แล้วกด `Next` ไปเรื่อยๆ จนจบการติดตั้ง

## ขั้นตอนที่ 3: โหลด MRA โปรเจกต์ไปติดตั้ง
1. บนเว็บ AI Studio แห่งนี้ ไปที่เมนู **ตั้งค่า (รูปเฟืองมุมขวาบน ⚙️)**
2. เลื่อนลงมาจะพบปุ่ม **"Export to ZIP"** ให้กดเพื่อดาวน์โหลดไฟล์
3. นำไฟล์ ZIP ที่ดาวน์โหลดมา ไปแตกไฟล์ (Extract Here / Extract to folder) ปลายทางไว้ที่ไดรฟ์ไหนก็ได้ที่คุณต้องการ เช่น `D:\MRA_System`

## ขั้นตอนที่ 4: การเปิดระบบครั้งแรก (เริ่ม Start Server)
1. เข้าไปในโฟลเดอร์ที่คุณแตกไฟล์เอาไว้
2. หาไฟล์ที่ชื่อว่า **`start_app.bat`** (ถ้าเครื่องคุณปิดนามสกุลไว้ มันจะชื่อ start_app และมีไอคอนรูปฟันเฟือง/หน้าจอคอม)
3. ดับเบิลคลิกไฟล์ได้เลย
   * จะมีหน้าต่างสีดำ (Command Prompt) เด้งขึ้นมา
   * มันจะทำการโหลดโค้ด/ปลั๊กอิน (ในครั้งแรกอาจรอ 1-2 นาที)
   * หากมีหน้าต่าง **Windows Security Alert (Firewall)** เด้งถาม ให้ติ๊กถูกทั้ง Private และ Public แล้วกด **"Allow access"**
4. หากทุกอย่างสมบูรณ์ หน้าต่างสีดำจะแสดงบรรทัด `Server running on http://0.0.0.0:3001`
5. และหน้าเว็บ MRA `http://localhost:3001` จะเด้งเปิดขึ้นมาในเบราว์เซอร์อัตโนมัติ

**หมายเหตุ**: 
- **ห้ามปิดหน้าต่างสีดำ! (Command Prompt)** ตราบใดที่ยังต้องการให้ระบบทำงาน (หากเผลอปิด ให้คลิก start_app.bat เพื่อเปิดใหม่)
- คุณสามารถลากไฟล์ `start_app.bat` ไปทำ Shortcut ไว้ที่หน้า Desktop ได้

## ขั้นตอนที่ 5: การให้เครื่องอื่นในโรงพยาบาล 들어มาใช้งาน
เครื่องที่คุณทำการ Start `start_app.bat` ครั้งแรก จะทำหน้าที่เป็น Server

หากคอมพิวเตอร์พยาบาลเครื่องอื่นๆ ในวอร์ด อยากเข้ามาคีย์ใบงาน:
1. ให้หาว่า ปัจจุบัน เครื่อง Server (เครื่องที่รันหน้าจอดำ) มี **IP ในวงแลน** IP อะไร (ตัวอย่าง: 192.168.1.55)
2. ไปที่เครื่องของพยาบาลวอร์ด เปิด Chrome 
3. พิมพ์ที่อยู่: `http://192.168.1.55:3001` แล้วกด Enter
4. จะสามารถเข้าใช้งานและ Audit ข้อมูลได้ทั้งหมด (ข้อมูลจริง!) ทันที
